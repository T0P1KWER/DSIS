#  Ветеринарная клиника (Veterinary Clinic)

Десктопное приложение с графическим интерфейсом для автоматизации работы регистратуры ветеринарной клиники. Реализовано на языке Python с использованием библиотеки `tkinter` и архитектурного паттерна **Model-View-Controller (MVC)**.

##  Общая архитектура

Приложение построено по принципу разделения ответственности (Separation of Concerns):

*   **Model (Домен):** Бизнес-сущности и хранение данных в памяти (класс `Pet` с типизированными датами).
*   **View (Представление):** Графический интерфейс на базе `tkinter` (окна, таблицы, диалоги).
*   **Controller (Логика):** Координация действий, обработка событий пользователя, управление состоянием.
*   **Infrastructure (Инфраструктура):** Работа с файлами данных формата XML с использованием парсеров **DOM** (запись) и **SAX** (чтение).

##  Структура проекта

```text
project_root/
