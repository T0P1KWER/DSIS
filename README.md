# DSIS Labs

Лабораторные работы по дисциплине "Структуры данных и информационный поиск"

## Содержание

### [Lab 1: Мультимножества](lab_multiset/)
- Реализация мультимножеств с операциями объединения, пересечения, разности
- Построение булеана
- Перегрузка операторов
- Тесты с покрытием кода

### [Lab 2: Машина Поста](lab_post_machine/)  
- Симулятор машины Поста
- Лента, головка, система команд
- Загрузка программ из файлов
- Тесты с покрытием кода

## Сборка и запуск

Каждая лабораторная имеет свои скрипты для сборки и тестирования.
Подробности в соответствующих папках.
